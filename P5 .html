Convolutioneel Neuraal Netwerk¶

Nu we het baseline model hebben weten we wat we moeten overtreffen en kunnen we een echt netwerk gaan bouwen. Ik maak gebruik van verschillende technieken. Batch normalization voert een transformatie uit waardoor de gemiddelde outputs dicht bij 0 of 1 blijven. Ook maak ik gebruik van de Dropout functie. De dropout functie zorgt ervoor dat er een bepaald procent van de neuronen niet mee doet tijdens die itteratie en voorkomt dus dat het netwerk afhankelijk wordt van een specefieke neuron. Ik heb het percentage op 20% gezet. MaxPooling vermindert de parameters door de hoogste waarde te kiezen dit zorgt er voor dat het model sneller is. In elke laag gebruik ik de relu als activatie functie. Die was het meest voorkomend en het meest gebruikt.

De output kan geen 3x3 matrix zijn dus gebruik ik de functie Flatten om er een 1 matrix van te maken. Ook zie je dat we in de laatste laag de softmax functie gebruiken. Dit gebeurt vaak bij classificatie modellen het output namelijk een voorspelling, een kans dat het die klasse is.

In [96]:
# De output kan geen 3x3 matrix zijn dus gebruiken we de functie flatten om er 1 matrix van te maken.
model.add(layers.Flatten())
model.add(Dropout(0.2))

model.add(layers.Dense(256, activation='relu'))
model.add(Dropout(0.2))
model.add(BatchNormalization())

# Dense zetten we op 10 omdat we 10 verschillende outputs (classes) willen
model.add(layers.Dense(10, activation='softmax'))

Nu het model compleet is kunnen we het gaat compile en fitten. Als optimizer heb ik adam gekozen want die wordt het meest gebruikt en is in de meeste gevallen het best. Het aantal epochs staat op 15.
